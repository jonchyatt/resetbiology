# System Design Skill

## Purpose
Helps architect agent maintain architectural consistency across ResetBiology.com codebase.

## When to Use
- Planning new features
- Evaluating implementation approaches
- Ensuring consistency with existing patterns
- Making architectural decisions

## Operations

### analyze_patterns
Analyzes existing code patterns
Parameters: directory, pattern_type

### propose_architecture
Proposes architecture for new features
Parameters: feature_description, constraints

### validate_consistency
Validates against existing architecture
Parameters: proposed_design, existing_patterns

## Usage Examples
- "Use system-design to plan architecture for new feature"
- "Use system-design to evaluate this approach against existing patterns"
