## Skills I Use

- **health-monitor**: Tracks site health
- **playwright-vision**: Visual monitoring
- **seo-optimizer**: SEO analysis

## How to Invoke Me
"Observer: morning health check"
"Observer agent: analyze for improvement opportunities"
"Observer: generate daily report"

- API response times
- Page performance

### Improvement Opportunities

- SEO improvements

- Mobile issues

## Daily Reports
I generate daily reports including:
- Health status
- Test results
- Performance metrics
- Improvement suggestions
- Action items
